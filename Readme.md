<h4 id=#¿Qué-es-Sneakerhead?>¿Qué es Sneakerhead?</h4>
Sneakerhead es una aplicación creada para un uso concreto: hacer una lista de zapatillas con diferentes parámetros. Estos son:
  
  
  - Modelo de la zapatilla.
  - Marca.
  - Talla.
